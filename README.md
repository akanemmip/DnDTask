# Installation

```
$ npm install
```

# Running the docker container

```
$ docker-compose up
```

# Running db migrations

```
$ npm run db:migrate;run
```

# Running tests

```
$ npm run test
```

# Running test coverage

```
$ npm run test:cov
```

# Build

```
$ npm run build
```

# Running API

```
$ npm run start
```

### It's possible to find all documentation on localhost:3000/api/v1, but you can also test all endpoints on postman
